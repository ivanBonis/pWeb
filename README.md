# Programação Web - Universidade da Beira Interior (UBI)

## Descrição do Repositório
Este repositório centraliza o código-fonte e a documentação técnica desenvolvida no âmbito da Unidade Curricular de Programação Web, lecionada na Universidade da Beira Interior (Covilhã). O projeto abrange os componentes práticos e laboratoriais da turma PL1.

## Tecnologias e Linguagens
O desenvolvimento assenta nas seguintes tecnologias padrão de arquitetura *front-end*:
* **HTML5** (80.5%): Estruturação semântica e marcação de hipertexto.
* **CSS3** (16.9%): Estilização, formatação em cascata e *design* responsivo.
* **JavaScript** (2.6%): Lógica de programação *client-side* e manipulação dinâmica do *Document Object Model* (DOM).
